Homework

In every week there will be the following homework. In some weeks there may be additional homework. All homework is for your own practice and learning. It will not be marked or graded.

Homework common to all weeks:

  • Review the material from the lecture(s) this week. Make sure you understand the concepts and ideas presented. Ensure you can follow any mathematical or statistical explanations. Ensure you can do the R that is needed for the week’s learning objectives.
  • When you find things you don’t understand or have trouble doing (e.g., in R), make a note of these. You can then ask a teaching assistant (TA) for help during the practical session, and you can ask on the Discussion Forum on OLAT. You can also ask questions during the lectures.

There is no other homework for this week.

Practical

Exercise 1 (of 1)

During the first lecture you collected data on class reaction times. The exercise this unit is for you to use R to analyse the dataset.

To help, we give you a nearly complete and working script of R code. The script does, however, contain some deliberate errors! You’ll need to find and correct these! This is a common feature of code we write – we usually make a mistake of some kind, get and error message, and need to troubleshoot and correct. So this exercise is good practice for what you’ll experience during the course!

The script also contains some “???” that you need to replace with appropriate text. The errors get a bit harder as you go through the script. Good luck!

Click on the links below, and hopefully the files will download. If they appears in your browser do Save As... and save them to somewhere safe on your computer. Do not add a file extension. Do not let a file extension be added (e.g., the Chrome browser automatically attempts to add a .txt file extension… you have to delete it!). Leave the file extension as they are (only .r or .csv) . Do not open the data file in Excel or any other program.

Script file: reaction_time_analysis.r

Data file: Link to Reaction Time Dataset

Get organised by setting up a project folder on your computer and downloading the above files into it. This will be important so you can know where your copy of the data files is located. You must know this to be able to read the data into R.

Then open the script in RStudio and get it working, not forgetting to think about the questions in the script, about assumptions, effect size, biological significance, statistical significance, and so on.

Write a sentence about the finding. Cover statistical and practical significance. Be critical. Ask a teaching assistant (TA) for feedback, see if they can make suggests of how you could improve what you’ve done.

The statistical test used in the script is a t-test. Hopefully you have experienced this in previous studies. Please ensure you have reviewed that material if you are not familiar with it anymore. If you are new to t-tests, please ask a TA for help.

If you finish early and are happy with everything so far, and you want a challenge, figure out how to use RMarkdown to create a word document, html file, or pdf of your findings.
